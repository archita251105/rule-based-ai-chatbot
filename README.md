# 📖 Overview

NovaMind is an intelligent browser-based chatbot that simulates conversational AI using a predefined knowledge base and rule-based intent classification.

Unlike API-powered chatbots, NovaMind works completely offline without requiring any external AI service, backend, or cloud infrastructure. It provides a smooth conversational experience with modern UI animations and several interactive features.

This project demonstrates strong front-end development skills, DOM manipulation, event-driven programming, UI/UX design, and JavaScript logic implementation.

# 💡 How It Works

The chatbot follows a simple NLP-inspired workflow:

```
User Input
      │
      ▼
Intent Classification
      │
      ▼
Knowledge Base Matching
      │
      ▼
Generate Response
      │
      ▼
Display Chat Bubble
      │
      ▼
(Optional)
Speech Synthesis
```
